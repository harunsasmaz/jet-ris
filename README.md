## Dependencies

    - SDL
    - SDL_gfx

## Install

    - brew install sdl
    - brew install sdl_gfx

## Compile

    - make

## Gameplay

<kbd>Z</kbd> Rotate piece

<kbd>X</kbd> Drop piece

<kbd>Left</kbd> Move piece to left

<kbd>Right</kbd> Move piece to right

<kbd>Down</kbd> Move piece to down

<kbd>Esc</kbd> Quit game
